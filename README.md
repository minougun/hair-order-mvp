# hair-order-mvp

美容室で要望を言語化しづらい人向けの、オーダー文メーカーMVPです。

## できること

- 現状・悩み・なりたい印象・制約を入力
- 20本のルールテンプレから最適候補を自動選択
- 次の4ブロックを生成
  - 要約
  - 具体指示
  - 避けたいこと
  - 美容師に確認すること
- 検索ワードの提案
- そのまま見せる本文のコピー
- SNS向けカード文のコピー
- 結果カードのPNG保存（1080x1350）

## 使い方

1. `index.html` をブラウザで開く
2. 項目を選んで `オーダー文を生成` を押す
3. 結果をコピーして美容師に提示する

ローカルサーバーで開く場合:

```bash
cd /mnt/c/Users/minou/hair-order-mvp
python3 -m http.server 8080
```

`http://localhost:8080` にアクセス。

## 設計方針

- MVPは画像解析なしで成立させる
- 推奨ロジックはルールベースで説明可能性を確保
- 医療・障害・性格の判定は行わない
- 出力は「診断」ではなく「提案・言語化支援」
